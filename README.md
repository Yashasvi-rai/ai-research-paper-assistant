# 🧠 AI Research Paper Assistant

An AI-powered research paper assistant built using RAG (Retrieval-Augmented Generation), FAISS, Ollama, and Streamlit.

## 🚀 Features

- Upload research paper PDFs
- Ask questions about papers
- AI-generated answers
- Automatic paper summarization
- Beautiful dashboard UI
- Local LLM support using Ollama

---

## 🛠️ Tech Stack

- Python
- Streamlit
- FAISS
- Sentence Transformers
- Ollama
- TinyLlama

---

## 📌 How It Works

1. Upload a research paper PDF
2. Text is extracted and chunked
3. Embeddings are generated
4. FAISS retrieves relevant chunks
5. TinyLlama generates answers

---

## ▶️ Run Project

```bash
streamlit run app.py
```
